# Lancify — Freelance Platform

A full-stack freelance marketplace where clients can 
post projects and freelancers can bid on them.


##  Tech Stack
- Frontend: React.js, CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT

##  Features
- User authentication (Login/Signup)
- Post and browse freelance projects
- Bid system for freelancers
- Dashboard for clients and freelancers
